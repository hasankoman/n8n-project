## Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/hasankoman/n8n-project.git
```

2. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

## Usage Guide

Visit `http://localhost:3000` to view the application.
